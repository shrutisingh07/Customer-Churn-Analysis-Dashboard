# Customer Churn Analysis & Prediction Dashboard

## Overview
This project analyzes customer churn data and predicts whether a customer is likely to leave a telecom company.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Churn Rate Analysis
- Contract Type Analysis
- Monthly Charges Analysis
- Machine Learning Prediction
- Interactive Streamlit Dashboard

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Streamlit

## Results
- Model Accuracy: 78.54%
- Highest Churn Rate: Month-to-month customers (42.71%)
- Lowest Churn Rate: Two-year contract customers (2.83%)
